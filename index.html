Projects

kutsix.com

It is a ride sharing platform. It works as follows-

Say a person having own vehicle goes to office alone. So the rest of the seats of the vehicle is blank. Now there maybe people from the vicinity area having the same destination but doesn't own any vehicle. So combining these two types of user is how kutsix works. The vehicle owner posts the ad of his ride and interested people request to his ride. Ride is confirmed when the vehicle owner accepts the request.

Languages, scripts, framework and database used for making of this application are :

    1.Frontend :
  • HTML5
  • CSS
  • JavaScript
  • JQuery
    2.Backend :
  • Golang
    3.Framework :
  • Revel : A Go web application framework
    4.Database :
  • Mongodb

Currently I am working on its android application
